# 智能客服与自动回访

## 原需求描述

部署AI语音机器人或文本机器人，自动回访意向客户、解答常见问题、预约看房时间。

---

## 页面结构

```
- 客服首页
  - 对话入口
  - 服务记录

- 在线咨询页
  - 智能问答
  - 转人工

- 回访管理页
  - 回访任务
  - 回访记录

- 知识库页
  - FAQ维护
  - 知识配置
```

## 功能列表

### 1. AI语音机器人

- 语音自动接听
- 语义理解
- 智能对话：
  ```
  [通话记录]

  来电时间：2024-01-15 14:30
  客户：138xxxx1234
  意图：咨询租金缴纳
  处理结果：已解答
  满意度：满意
  ```

### 2. 文本机器人

- 常见问题解答：
  ```
  [在线咨询]

  用户：租金怎么交？
  AI：租金可通过以下方式缴纳：
  1. 小程序在线支付
  2. 银行转账至账户：XXX
  3. 到管理中心现场缴纳

  如需帮助，请回复"人工"
  ```

### 3. 意向客户自动回访

- 回访任务配置
- 自动外呼
- 回访记录：
  ```
  [回访记录]

  客户：张三
  意向：看房预约
  回访时间：2024-01-15 15:00
  回访结果：已确认周六上午看房
  跟进状态：[已确认]
  ```

### 4. 看房预约

- 预约时间选择
- 预约确认
- 预约提醒

### 5. 常见问题库

- FAQ配置
- 自动匹配回复
- 知识学习优化

## 数据字段

| 字段 | 类型 | 说明 |
|------|------|------|
| session_id | string | 会话ID |
| customer_name | string | 客户姓名 |
| phone | string | 电话 |
| intent | string | 意图 |
| response | string | 回复内容 |
| satisfaction | int | 满意度 |
